# dotfiles
Personal dotfiles for my Arch + Hyprland setup

## Packages


```
hyprland            // window manager
wofi                // application launcher
mako                // notification daemon
nemo                // file browser
kitty               // terminal
fish                // shell
omf                 // fish utilities wrapper
rog-control-center  // asus rog utils
nwg-panel           // status bar
appimagelauncher    // appimage installer
ark                 // archive manager
pavucontrol         // sound control
pipewire            // multimedia framework
wireplumber         // manager for pipewire
systemd             // unit orchestrator
```
